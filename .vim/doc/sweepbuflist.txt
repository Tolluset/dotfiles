*sweepbuflist.txt*	Plugin for sweeping the buffer list
                                                Last change: 2010 Jan 14.

SYNOPSIS					*sweepbuflist-synopsis*
>
	" only sweeping
	:SweepBufList

	" with showing status by ":buffers", hit <Leader>sb

DESCRIPTION					*sweepbuflist-description*

Plugin to delete buffers that are listed but unloaded from the buffer list.

PROVIDES					*sweepbuflist-provides*

Mappings:
<Leader>sb or <Plug>SweepBufList
	Sweep and then show the buffer list.

Commands:
:SweepBufList
	Sweep the buffer list.

SETTINGS					*sweepbuflist-settings*

This plugin doesn't have any settings.

DISABLES					*sweepbuflist-disables*

Variables:
g:loaded_sweepbuflist
	When exists, does nothing.  Use to disable this plugin.

g:no_sweepbuflist_maps
	When non-zero, doesn't register any effective mappings.

g:no_plugin_maps
	When non-zero, doesn't register any effective mappings.  This variable
	may have an affect on mappings of other plugins.  Use
	"g:no_sweepbuflist_maps" to disable only mappings of this plugin.
