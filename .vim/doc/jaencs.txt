*jaencs.txt*	Plugin for "fileencodings" configuration
                                                Last change: 2010 Jan 07.

DESCRIPTION						*jaencs-description*

Plugin for setting 'fileencodings' appropriate for Japanese that is depending
on 'encoding' automatically.  Thus changing "encoding" is a trigger of this
plugin.  Alike when you start up VIM, "fileencodings" is setted.

It is made sure to work with following encodings.

	utf-8
	cp932, sjis
	euc-jp, euc-jisx0213, eucjp-ms
	iso-2022-jp, iso-2022-jp-3

DEPENDENCIES						*jaencs-dependencies*

{only available when your VIM has the |+multi_byte| and |+autocmd| feature}

PROVIDES						*jaencs-provides*

Augroups:
jaencs
	All of the autocmds provided by this plugin is located in "jaencs".
	To disable in a certain instance of VIM, hit the following command: >

		:autocmd! jaencs

SETTINGS						*jaencs-settings*

This plugin doesn't have any settings.

DISABLES						*jaencs-disables*

Variables:
g:loaded_jaencs
	When exists, does nothing.  Use to disable this plugin.
