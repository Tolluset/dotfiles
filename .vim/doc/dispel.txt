*dispel.txt*	Plugin for showing invisible characters
                                                Last change: 2010 Jan 13.

DESCRIPTION						*dispel-description*

Plugin for showing invisible characters automatically.

DEPENDENCIES						*dispel-dependencies*

{only available when your VIM has the |+autocmd| feature}

PROVIDES						*dispel-provides*

Augroups:
dispel
	All of the autocmds provided by this plugin is located in "dispel".
	To disable in a certain instance of VIM, hit the following command: >

		:autocmd! dispel

SETTINGS						*dispel-settings*

DEFAULT

In the default settings - when "g:dispel_patterns" is undefined - this plugin
show trailing spaces and U+3000 "IDEOGRAPHIC SPACE" with Error highlighting.

Variables:
g:dispel_patterns
	Patterns and highlighting group names of characters to show.  This
	must be a List has a/some Dictionar(y|ies).  Dictionary must have
	following keys and values.

	key		meaning ~
	name		A name of pattern.  Must be unique through all pattern
			names.
	pattern		A |pattern|.  Can be written in the form of regular
			expression.
	highlight	A name of highlighting group.  Must be the one that
			exists already.

	When you wanna show trailing spaces with WarningMsg highlighting,
	write following codes in your .vimrc: >

		let g:dispel_patterns = [
		        \   {
		        \       'name':      'TrailingWhiteSpace',
		        \       'pattern':   '\s\+$',
		        \       'highlight': 'WarningMsg'
		        \   },
		        \ ]
<
	Note that overwriting default patterns.  To add patterns newly, use
	the variable "g:dispel_add" (see below).

	Note that when this script is sourced, your configuration become
	effective.  Thus only changing this variable from command line is no
	point.  In order to change settings, write new values in .vimrc and
	restart VIM for now.

g:dispel_add
	When evaluate to non-zero, default patterns are used in addition to
	specified by "g:dispel_patterns".  Following pattern is useful when
	writing docs: >

		let g:dispel_add = 1
		let g:dispel_patterns = [
		        \   {
		        \       'name':      'ASpaceBetweenSentences',
		        \       'pattern':   '\v\.@<= \w@=',
		        \       'highlight': 'Error'
		        \   },
		        \ ]

DISABLES						*dispel-disables*

Variables:
g:loaded_dispel
	When exists, does nothing.  Use to disable this plugin.
