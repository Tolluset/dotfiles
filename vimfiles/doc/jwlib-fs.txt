*jwlib-fs.txt*	Autoload script about filesystem
                                                Last change: 2010 Jan 01.

SYNOPSIS						*jwlib-fs-synopsis*
>
	let cmd = 'write'
	if fs#Exists(expand('%:p'))
	    cmd .= '!'
	endif

DESCRIPTION						*jwlib-fs-description*

Autoload script about filesystem.  Here there are descriptions its name start
with "fs", specifically following files.

autoload/fs.vim
			Contains a function to know a existence of a file.

PROVIDES						*jwlib-fs-provides*

Functions:
fs#Exists({file})
			When {file} exists, return not zero.  Otherwise return
			zero.
			This function is contained in autoload/fs.vim
