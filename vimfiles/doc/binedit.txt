*binedit.txt*	Plugin for editing binary
                                                Last change: 2010 Jan 04.

SYNOPSIS						*binedit-synopsis*
>
	:edit ++bin binary.bin

DESCRIPTION						*binedit-description*

Plugin for editing binary in the form of "xxd" when you open a file with
the option 'binary'.  To set the 'binary' with opening a file simultaneously,
use the |++bin| modifier.  See |using-xxd| about the source of this plugin.

SETTINGS						*binedit-settings*

This plugin doesn't have any settings.
