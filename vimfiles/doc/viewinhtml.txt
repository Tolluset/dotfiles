*viewinhtml.txt*	Plugin for showing the file in html
                                                Last change: 2010 Jan 16.

SYNOPSIS						*viewinhtml-synopsis*
>
	" all of the buffer
	" hit <Leader>vh or
	:ViewInHtml

	" first 10 lines
	:1,10ViewInHtml

DESCRIPTION					*viewinhtml-description*

Plugin to show contents of the current buffer in your favorite http user agent
(web browser).

DEPENDENCIES					*viewinhtml-dependencies*

{only available when your VIM has the |+multi_byte| and |+win32| feature}

This plugin require following autoload file.

autoload/jwlib/shell.vim
http://github.com/januswel/dotfiles/blob/master/vimfiles/autoload/jwlib/shell.vim

The command ":TOhtml" is also needed but it is distributed as a standard
plugin.

PROVIDES						*viewinhtml-provides*

Mappings:
<Leader>vh or <Plug>ViewInHtml
	Shows all contents of the current buffer in system-default web
	browser.  It is convenient for hard copy when there is a difference
	between a value of 'encoding' and valid one for your printer.

Commands:
:[range]ViewInHtml
	Same as <Leader>vh or <Plug>ViewInHtml but can be specified [range].

SETTINGS						*viewinhtml-settings*

This is a wrapper for ":TOhtml", so its settings are available.  See
|:TOhtml|.

DISABLES						*viewinhtml-disables*

Variables:
g:loaded_viewinhtml
	When exists, does nothing.  Use to disable this plugin.

g:no_viewinhtml_maps
	When non-zero, doesn't register any effective mappings.

g:no_plugin_maps
	When non-zero, doesn't register any effective mappings.  This variable
	may have an affect on mappings of other plugins.  Use
	"g:no_viewinhtml_maps" to disable only mappings of this plugin.
